## 출력 확인 시 주의사항

일부 코드의 `printf()` 서식 문자열에는 숫자 사이 공백이 포함되어 있지 않습니다.

예를 들어 다음 코드는:

```c
printf("%d%d\n", 6, 5);
```

아래처럼 출력됩니다.

```text
65
```

`6 5`처럼 공백을 넣고 싶다면 서식 문자열에도 공백을 작성해야 합니다.

```c
printf("%d %d\n", 6, 5);
```
