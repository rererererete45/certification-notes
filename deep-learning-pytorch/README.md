# 딥러닝과 PyTorch

딥러닝의 기본 원리와 PyTorch를 이용한 신경망 구현, 학습, 평가 과정을 정리합니다.

## 학습 분야

* 딥러닝 기초
* 텐서
* 자동 미분
* 신경망 구조
* 활성화 함수
* 손실 함수
* 최적화 알고리즘
* 학습률
* 과적합 방지
* 완전연결 신경망
* 합성곱 신경망
* 순환 신경망
* 모델 저장 및 불러오기

## 사용 도구

* Python
* PyTorch
* NumPy
* Matplotlib
* Jupyter Notebook

## 정리 방식

* 수학적 개념과 코드 연결
* 텐서의 형태와 연산 결과 확인
* 모델 구조와 학습 과정 기록
* 손실값과 평가 지표 시각화
* 오류와 해결 과정 기록
* 완성된 모델은 별도 프로젝트 저장소로 분리
