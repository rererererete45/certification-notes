# QGIS와 PyQGIS

QGIS 및 PyQGIS를 활용한 공간데이터 처리, 분석, 시각화 및 반복 작업 자동화 과정을 정리합니다.

## 학습 분야

* QGIS 기본 기능
* 공간데이터 구조
* 좌표계와 좌표 변환
* 벡터 데이터 처리
* 래스터 데이터 처리
* 공간 선택 및 공간 조인
* 버퍼 분석
* 중첩 분석
* 지오프로세싱
* 지도 시각화
* PyQGIS 기본 문법
* QGIS 작업 자동화

## 사용 도구

* QGIS
* PyQGIS
* Python
* GeoPandas
* Shapely

## 정리 방식

* 입력자료와 좌표계 기록
* 분석 목적과 처리 순서 작성
* QGIS 수작업과 PyQGIS 자동화 방식 비교
* 코드와 실행 결과 함께 기록
* 지도 및 분석 결과 이미지 첨부
* 완성된 공간분석은 별도 프로젝트 저장소로 분리
