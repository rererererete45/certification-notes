# 머신러닝

머신러닝의 주요 알고리즘, 데이터 전처리, 모델 학습, 성능 평가 및 개선 과정을 정리합니다.

## 학습 분야

* 머신러닝 기본 개념
* 지도학습과 비지도학습
* 데이터 전처리
* 특성 선택 및 특성 공학
* 회귀
* 분류
* 군집화
* 차원 축소
* 모델 평가
* 교차검증
* 하이퍼파라미터 튜닝
* 과적합과 일반화

## 사용 도구

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## 정리 방식

* 알고리즘의 원리와 사용 조건 정리
* 데이터 전처리 과정 기록
* 기준 모델과 개선 모델 비교
* 평가 지표 선택 근거 작성
* 학습 결과와 오류 분석
* 완성된 모델은 별도 프로젝트 저장소로 분리
