# 통계학

데이터 분석과 머신러닝에 필요한 통계 개념, 공식, 해석 방법과 실습 내용을 정리합니다.

## 학습 분야

* 기술통계
* 확률
* 확률분포
* 표본과 모집단
* 추정
* 가설검정
* 상관분석
* 회귀분석
* 분산분석
* 비모수 통계

## 정리 방식

* 개념과 용어 정리
* 공식의 의미와 적용 조건 확인
* 계산 과정을 단계별로 기록
* Python을 이용한 통계 실습
* 결과표와 통계량 해석
* 자주 혼동하는 개념 비교
