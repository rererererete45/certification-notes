# SQL

## 학습 분야

* 데이터베이스 기초
* SELECT
* WHERE
* GROUP BY
* HAVING
* ORDER BY
* JOIN
* 서브쿼리
* 집합 연산
* 윈도 함수
* 데이터 정의어
* 데이터 조작어
* 데이터 제어어
* 트랜잭션

## 정리 방식

* 문법과 실행 순서 정리
* 예제 테이블을 이용한 쿼리 작성
* 입력 데이터와 실행 결과 함께 기록
* 유사한 문법 비교
* 오류가 발생한 쿼리와 수정 과정 기록
* 데이터 분석에 활용되는 쿼리 중심 정리
