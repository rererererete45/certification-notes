# 지적기사 실기

지적기사 실기시험을 준비하며 계산문제, 측량 절차, 성과 작성 방법, 기출문제와 오답을 정리합니다.

## 학습 분야

* 지적측량 계산
* 경위의측량
* 평판측량
* 면적측정
* 좌표면적 계산
* 교회법
* 도선법
* 측량성과 작성
* 작업형 절차
* 기출문제 풀이

## 정리 방식

* 공식과 계산 원리 정리
* 풀이 과정을 단계별로 기록
* 단위와 반올림 기준 확인
* 자주 틀리는 계산 유형 분류
* 작업 순서와 주의사항 정리
* 기출문제와 오답 반복 학습
